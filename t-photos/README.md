# Vehicle Photos

Photos of the robot from all required angles.

## Views

**Front view**
Shows the color sensor facing down and the 
front distance sensor facing forward.

**Back view**
Shows the traction motor and rear wheel 
configuration with both wheels on the same axle.

**Left view**
Shows the left distance sensor placement 
on the side of the robot.

**Right view**
Shows the right distance sensor placement 
on the side of the robot.

**Top view**
Shows the Spike hub, all port connections 
and overall robot layout.

**Bottom view**
Shows the wheel configuration and Ackermann 
steering linkage underneath.
